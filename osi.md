# layer 7 (application)
- user interaction with data
- ex: https, dhcp

# layer 6 (presentation)
- translator and formatter of data 

# layer 5 (session)
- responsible for managing and controlling sessions between two applications
- ex: rtcp, pptp

# layer 4 (transport)
- end to end communication between different hosts
- tcp and udp : manage data delivery and error checking
 
# layer 3 (network)
- logical addressing and routing 
- choses best path for data to sent
- ex: ospf, rip

# layer 2 (data link)
- local delivery + mac address + error check 
- nic comes with every networked device 

# layer 1 (physical)
- all hardware and cable connections
